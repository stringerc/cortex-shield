How to Report a Bug

  1. Visit GitHub Issues
  2. Click "New Issue"
  3. Include: the website URL, what AI element wasn't blocked, and your mode (Ghost/Sentry/Guardian)

How to Suggest a New AI Rule

If you find an AI widget that CortexShield doesn't catch:

  1. Open a GitHub Issue with the label new-rule
  2. Include the website URL and the AI element you see
  3. We'll add a static rule and release an update
